> v3

High performance topic based reliable brokerless UDP messaging ("event-bus"). Up to 7 million 70 byte messages per second. Buffers required to implement NAK protocol are kept offheap, so there is little impact on GC. Reliable, bounded Low Latency if configured according.

See https://github.com/RuedigerMoeller/fast-cast
